═══════════════════════════════════════════════════════════════
  🚀 PRÓXIMOS PASSOS APÓS BAIXAR
═══════════════════════════════════════════════════════════════

1. EXTRAIR O ZIP
   - Descompacte o arquivo no seu computador

2. INSTALAR DEPENDÊNCIAS
   npm install

3. DEPLOY NA NETLIFY
   Opção A: Conecte o repositório no GitHub → Netlify
   Opção B: Use Netlify CLI
      npm install -g netlify-cli
      netlify deploy --prod

4. CONFIGURAR VARIÁVEIS DE AMBIENTE (no painel da Netlify)
   DATABASE_URL=sua_url_postgresql
   INVICTUS_LINK_BASICO=https://go.invictuspay.app.br/...
   INVICTUS_LINK_COMPLETO=https://go.invictuspay.app.br/...
   INVICTUS_LINK_PREMIUM=https://go.invictuspay.app.br/...
   INVICTUS_LINK_REDACAO=https://go.invictuspay.app.br/...
